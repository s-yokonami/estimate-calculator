# 見積金額・工数計算ツール

工数単価・時間を入力すると見積金額・利益率を自動計算するツールです。  
**社内専用ツールです。社外への公開・共有はしないでください。**

## 機能

- 工数単価 × 時間から見積金額を自動計算
- 販売見積金額・制作費・見込利益の表示
- 等級別工数分析（等級ごとの予算内工数・見込利益を比較）
- 設定の配布用URL生成（利益率・等級単価を社内に配布）
- 入力内容の共有URL生成（設定込みで社内共有）

## セットアップ

```bash
npm install
npm run dev
```

## ビルド

```bash
npm run build
```

## 設定について

右上の「設定」ボタンから利益率・等級別時給を設定できます。  
「配布用URLを生成」で生成したURLを社内メンバーに渡すと、URLを開くだけで設定が自動適用されます。

> ⚠ 設定URLには利益率・等級単価が含まれます。**社外には共有しないでください。**

## 技術構成

- Vanilla JS（フレームワークなし）
- [Vite](https://vitejs.dev/)
- 設定・入力値は `localStorage` に保存（外部サーバーへの送信なし）
